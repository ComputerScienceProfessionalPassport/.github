## 🚀 Our mission

**Computer Science Professional Passport (CSPP)** makes high-quality computer
science education open and accessible to everyone. We turn scattered, intimidating
material into clear, hands-on learning paths — so anyone can go from the
fundamentals to job-ready skills, for free.

## 🧭 What we're building

- **📚 CSPP Resources** — a free, open learning library: structured tutorials and
  references organized by track — Computer Fundamentals, Python, JavaScript, PHP,
  Data Structures & Algorithms, and more.
- **🖥️ CSPP Platform** — the full-stack application that powers the experience: a
  NestJS API, a React web app, and the Docusaurus resources site.
